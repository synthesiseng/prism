Pie Chart is the minimal canvas/data example adapted from the WICG HTML-in-Canvas repo.

It demonstrates coordinate helpers and basic surface composition without the larger application structure of Prism Atlantic or React Composer Lite.

Use this example to see the smallest practical shape of canvas drawing plus Prism-managed HTML/CSS surfaces.
