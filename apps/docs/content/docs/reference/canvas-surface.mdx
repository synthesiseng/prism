---
title: CanvasSurface
description: Registered HTML element managed by a CanvasRuntime.
---

`CanvasSurface` represents a registered HTML element. Apps receive it from `runtime.registerSurface()`.

Do not construct `CanvasSurface` directly.

```ts
const surface = runtime.registerSurface(element, {
  bounds: { x: 0, y: 0, width: 320, height: 180 }
});
```

## Methods

- `setBounds(bounds)`
- `dispose()`

## Bounds

Surface bounds use CSS pixels.

```ts
surface.setBounds({
  x: 24,
  y: 32,
  width: 280,
  height: 160
});
```

## Lifecycle

Use `runtime.unregisterSurface(surface)` or `surface.dispose()` when a surface leaves the runtime.

Prism handles DOM ownership restoration and cleanup for registered surfaces.
