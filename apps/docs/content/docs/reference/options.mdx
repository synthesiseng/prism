## CanvasRuntimeOptions

```ts
type CanvasRuntimeOptions = {
  backend?: "auto" | "native" | "fallback";
};
```

Use `"auto"` for normal code. Native is preferred when available; fallback is compatibility-only and lower fidelity.

## SurfaceOptions

```ts
type SurfaceOptions = {
  bounds: SurfaceBoundsInput;
  ariaLabel?: string;
};
```

## SurfaceBoundsInput

Bounds are CSS pixels. A surface can use a static bounds object or a function
that returns the current bounds.

```ts
type SurfaceBoundsInput =
  | {
      x: number;
      y: number;
      width: number;
      height: number;
    }
  | (() => {
  x: number;
  y: number;
  width: number;
  height: number;
});
```

## Handlers

```ts
type PaintHandler = (event: {
  ctx: CanvasRenderingContext2D;
  time: {
    now: number;
    delta: number;
    frame: number;
  };
  drawSurface(surface: CanvasSurface): void;
  invalidate(): void;
}) => void;

type UpdateHandler = (time: {
  now: number;
  delta: number;
  frame: number;
}) => void;
```

`now` and `delta` are seconds. `frame` is a monotonic frame counter.
