---
title: Paint Lifecycle
description: Prism owns invalidation, paint execution, surface drawing, and one-shot paint readiness.
---

Prism owns the runtime paint lifecycle. Apps describe what should happen during a paint pass.

```ts
runtime.onPaint(({ ctx, drawSurface }) => {
  drawCanvasScene(ctx);
  drawSurface(surface);
});
```

`drawSurface(surface)` is immediate. The order of calls is the composition order.

## Invalidation

Call `runtime.invalidate()` when app state changes and another paint pass is needed.

```ts
state.selectedId = nextId;
runtime.invalidate();
```

## Readiness

`paintOnce()` waits for one Prism-owned paint pass. It does not export image data.

```ts
await runtime.paintOnce();
canvas.toBlob((blob) => {
  // blob is produced by the normal canvas API.
}, "image/png");
```
