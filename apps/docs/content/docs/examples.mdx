## Prism Atlantic

Prism Atlantic is a vanilla TypeScript example using real NOAA/NHC HURDAT2 storm-track data.

It draws hurricane tracks on canvas and uses Prism-managed HTML/CSS surfaces for:

- title
- basin overview
- Saffir-Simpson legend
- tooltip
- detail panel
- caption
- export button

It demonstrates Prism in a data-viz canvas app: dense canvas rendering with rich HTML/CSS interface surfaces composed through the runtime.

## React Composer Lite

React Composer Lite is a React example where React-authored HTML/CSS components become movable, transformable, exportable Prism surfaces.

It demonstrates framework integration and the app/runtime boundary:

- React owns component state and UI.
- The app owns transforms and document state.
- Prism owns surface registration, paint readiness, and canvas composition.
